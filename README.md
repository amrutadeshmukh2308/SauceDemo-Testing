# SauceDemo-Testing
This project is an end-to-end automation testing framework built for the SauceDemo application (https://www.saucedemo.com/). It demonstrates a professional approach to test automation using Java, Selenium WebDriver, and TestNG.

🚀 Project Features
Framework Design: Page Object Model (POM) pattern for better maintenance and readability.

Complex Scenarios: Handles Login, Hamburger Menu navigation, About page validation, Product selection, Cart management, and Checkout flow.

Test Management: Integrated with TestNG for execution and test case organization.

Navigation Handling: Implements browser back-navigation and state management.

🛠 Tech Stack
Language: Java

Automation Tool: Selenium WebDriver

Test Runner: TestNG

Build Tool: Maven

IDE: IntelliJ IDEA

📂 Project Structure
src/test/java/pages/: Contains Page Objects (LoginPage, ProductPage, CheckoutPage).

src/test/java/tests/: Contains test suites and E2E flows.

pom.xml: Manages project dependencies.

⚙️ How to Run
Clone this repository to your local machine.

Ensure you have Java JDK and Maven installed.

Open the project in your favorite IDE.

Run the test suite using: mvn test
